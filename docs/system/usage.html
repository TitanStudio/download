如何使用？

点击屏幕左上角，即可呼出搜索栏，此搜索栏对于一些常用的网站可以直达官网， 省去了搜索引擎检索的步骤。
左上角热键是透明的，不影响视觉，只有在允许魔枢后台运行时才会生效。

快捷搜索

点击屏幕左上角，即可呼出搜索栏，此搜索栏对于一些常用的网站可以直达官网， 省去了搜索引擎检索的步骤。 左上角热键是透明的，不影响视觉，只有在允许魔枢后台运行时才会生效。

电脑管家功能

NEXUS内含软媒魔方电脑优化组件，可以实现清理垃圾、开机加速、WIFI共享等电脑管家所具有的功能。

激活工具

NEXUS内置了很多常用软件的激活工具，如Win7/8/10激活工具、MS Office激活工具、Adobe CS/CC激活工具、Corel激活工具等等。

VPN代理

NEXUS内置了一些VPN代理工具，方便实现科学上网。

快捷启动

将常用软件快捷方式发送到魔枢快捷启动即可在魔枢搜索框里启动软件，不仅限于软件，所有能创建快捷方式的图片、视频、文档、网页都可以。

工具箱

NEXUS工具箱不仅含有常用的小工具，还有系统常用功能的快捷入口。

软件库

精心挑选的软件以及下载地址，不用担心在假冒网站下载到掉包程序。如果是收费软件一般会推荐绿色版的下载。
