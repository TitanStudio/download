Visual Studio

应用程序的集成开发环境
Microsoft Visual Studio（简称VS）是美国微软公司的开发工具包系列产品。VS是一个基本完整的开发工具集，它包括了整个软件生命周期中所需要的大部分工具，如UML工具、代码管控工具、集成开发环境(IDE)等等。所写的目标代码适用于微软支持的所有平台。
官方网站：@Microsoft Inc.
适用平台：Windows

Dash

API文档浏览器
Dash是一个API文档浏览器（ API Documentation Browser），以及代码片段管理工具（Code Snippet Manager）。它就只有这两个功能，但确实是程序员最为关心的特性，可以毫不夸张地说，Dash是它们之中做的最好的一个！非常实用。从它第一版发布用到现在，绝对是一个你值得拥有的文档管理工具。
官方网站：@kapeli
适用平台：macOS/iOS

Atom

支持几乎所有格式的文本和代码编辑
Atom是一个十分强大的文本编辑器，可以编辑几乎所有格式的纯文本和源文件，可以安装插件定制更多功能、关键词高亮、甚至还可以自动补全你所打过的关键词。
官方网站：@atom
官方GitHub：@atom
适用平台：Windows/macOS/Linux
